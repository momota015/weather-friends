# Weather Friends ☀️
天気予報を楽しく確認できることをコンセプトに制作した天気アプリです。
一人暮らしの方や主婦の方が日常的に使うことを想定し、天候に応じて背景や動物キャラクターが変化する演出を取り入れ、見るたびに和める体験を目指しました。

## Features
- 都市名による天気検索機能
- 気温・湿度・天候アイコン・降水確率・最高/最低気温の表示
- 天候に応じた背景デザインと動物キャラクターの切り替え
- ローディング表示・エラーメッセージの実装
- Enterキーでの検索対応
- タイトルクリックで初期状態にリセット
- 検索回数に応じたダミー広告の表示
- PC・スマホ両対応のレスポンシブデザイン

## Challenges
- API連携が初めての挑戦だったため、Webで情報を調べながら`fetch`を使った天気データの取得と画面表示を実装
- 表示内容を充実させるため、現在の天気を取得するAPIと予報を取得するAPIを連続で呼び出し、降水確率や最高・最低気温まで表示する処理を実装
- 天候とキャラクターを連動させる関数を実装し、APIから取得した天候(Clear / Clouds / Rain)に応じて、背景クラスと動物画像を切り替える処理を作成
- ユーザー体験を考えた処理として、API通信中のローディング表示、空入力時や該当する都市が見つからない場合のエラーメッセージなど、想定される状況を考慮した実装を行った
- 広告表示の制御として、初回表示後に閉じた広告は、再度すぐ表示されると体験を損なうため、3回検索後に再表示される仕組みをJavaScriptで実装
- 背景の切り替えや動物のふわふわ動くアニメーションをCSSで実装することで、見ているだけで楽しい雰囲気を演出

## What I Learned
- `fetch`を使ったAPI連携と、取得したデータを画面に表示するまでの流れ
- 機能を作るだけでなく、「ユーザーがどう感じるか」を考えて実装内容を判断する視点(ローディング表示、エラー処理、キーボード操作対応など)
