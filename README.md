# Weather Friends

天気予報を楽しく確認できることをコンセプトに、天候に応じて背景や動物が変化する演出や検索回数に応じた広告表示機能を実装したWebアプリです。

## Features

- 都市名による天気検索
-  気温・湿度・天候アイコンの表示

## Challenges

- API連携を初めて実装
- 天候に応じて背景デザインと画像を変更
- 3回検索でダミー広告を表示する仕組みを実装
- JavaScriptで広告の表示・非表示を制御
- PC・スマホ両対応のレスポンシブデザイン

## Technologies

- HTML
- CSS
- JavaScript
- OpenWeather API
